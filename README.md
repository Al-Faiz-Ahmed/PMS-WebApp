## Installation

Clone Github's Repo on your local machine.

```
 git clone https://github.com/sheikhfaiz546/PMS-WebApp.git PMS-WebApp 
```
Install all Packages & Dependencies
```
 cd PMS-WebApp 
 
 npm install
```
Run on Local Host [http://localhost:3000](http://localhost:3000)
```
 npm start
```
